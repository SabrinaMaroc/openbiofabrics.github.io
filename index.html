Plateforme de documentation collaborative sur les biofabrics

Chez Open Biofabrics nous plussoyons l'open-source, c'est la raison pour laquel le contenu de ce site est placé sous différentes licences creative commons, sauf mention contraire. Ce qui veut dire que vous pouvez légalement partager, remixer et, réutiliser le contenu de cette plateforme tant que vous respecter les licences mentionnées. So SHARE, FORK, HACK :-)
